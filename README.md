# OliveBakery
친구 어머니의 빵집 예약 웹어플리케이션

springMVC mybatis mysql jsp html css bootstrap


기능

* 로그인 로그아웃 - 세션처리
* 공지사항
* 빵 주문 예약
* 빵 메뉴 조회
* mypage - 주문내역, 주소지 등록, 개인정보 변경
